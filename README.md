# PV_monitor

Dieses Respository beinhaltet Code und Hardware für den "Bau" des PV-Monitors des Freilabs. Dieses soll die aktuelle Leistung der Freilab-Photovoltaik-Anlage auf einer analogen Messuhr live darstellen.
Das Projekt soll realisiert werden durch eine eigens gefräste Platine sowie einem ESP32, der vom PV-Shelly die Daten abgreift.
